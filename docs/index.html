Full Setup

One command installs all skills, workflows, and hooks into your chosen IDEs.

$ npx agents-skills
┌ 🤖 Skill Installer
◇ Install where?
▸ Local Workspace
Global (~/.config)

✔ Antigravity
✔ VS Code / GitHub Copilot
Cursor
Claude Code
└ ✅ All done!

Install a Single Skill

Pick only what you need. The CLI detects your IDEs and copies to the right place.

$ npx agents-skills add react-query
◇ Install where? → Local Workspace
◇ Which IDEs? → Antigravity, Cursor
✅ Skill "react-query" installed into: Antigravity, Cursor
